# Exercise — Variables and Data Types

**Module 1** · Pre-lab practice · then open [`../../lab1/README.md`](../lab1/README.md)

## Goal

Create `Variables.java` with local variables of several primitive types and one `String`; print each.

## Do this

- Declare `int`, `long`, `double`, `boolean`, `char`, `String`
- Print with `System.out.println`
- `javac Variables.java && java Variables`

## Expected result

All declared values print without compile/runtime errors.

- [ ] Code compiles and runs (or notes complete if analysis-only)
- [ ] You can explain the result in one sentence
