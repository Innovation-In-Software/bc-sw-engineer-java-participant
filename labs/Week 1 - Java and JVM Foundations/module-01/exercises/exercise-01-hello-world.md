# Exercise — Hello World

**Module 1** · Pre-lab practice · then open [`../../lab1/README.md`](../lab1/README.md)

## Goal

Write, compile, and run a minimal program that prints `Hello, JVM!`.

## Starter / reference

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, JVM!");
    }
}
```

## Do this

- Save as `Hello.java`
- `javac Hello.java`
- `java Hello`
- Optional: `javap -c Hello`

## Expected result

Console prints `Hello, JVM!`; `Hello.class` exists.

- [ ] Code compiles and runs (or notes complete if analysis-only)
- [ ] You can explain the result in one sentence
